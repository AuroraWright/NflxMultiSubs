NflxMultiSubs
=============
全球首款支援 Netflix 全語言雙字幕的 Chrome 擴充套件，提供您更佳的觀影體驗！


特點
----
- 坊間首款支援日語等語言 (image-based) 的擴充套件
- 智慧選擇雙語字幕：看日本動畫顯示日語，看美劇顯示英文
- 整合原生 Netflix 選單，不需離開播放介面即可切換語言
- 在 Chrome 啟用 1080p 高畫質播放
